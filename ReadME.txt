## Проект "Pics to Words"

### ОСНОВНАЯ ИДЕЯ:

Основной цель - создание интерактивной игры для изучения иностранных языков через визуализацию слов. Пользователи могут учить новые слова, соединяя изображения с соответствующими словами, что способствует более эффективному запоминанию. Пользователь выбирает карточку с изображением, а затем соотносит ее с карточкой, содержащей слово. Цель игры — набрать как можно больше очков, собрав правильные пары карточек.

### ОСНОВНЫЕ КОМПОНЕНТЫ ПРОЕКТА:

1. **Начальный экран: здесь представлены правила игры и выбор языка для изучения.

2. **Игровой экран: отображает карточки для выбора, текущую статистику и шкалу здоровья.

3. **Экран результатов: показывает таблицу результатов и предлагает начать новую игру.


### ИСПОЛЬЗУЕМЫЕ ТЕХНОЛОГИИ:

- **Create React App: для быстрого старта проекта и настройки окружения.
- **React Router DOM: для навигации между различными экранами приложения.
- **Tailwind CSS: для стилизации интерфейса с применением гибкой системы классов.
- **@dubaua/get-declension: для корректного склонения числительных.

### ОСОБЕННОСТИ ПРОЕКТА:

-Интерактивность: переход между экранами без перезагрузки страницы.
-Адаптивность: игра подходит для пользователей разных возрастных категорий.
-Окончание игры и результаты: игра завершается при достижении трех ошибок или после сбора всех пар карточек. Модальное окно сообщает о результате: "Поражение" при трех ошибках и указывает количество собранных пар, "Победа" при успешном завершении всех карточек.
-Жизни и ошибки: пользователю предоставляется три "жизни". Каждая допущенная ошибка отнимает одну "жизнь". Этот элемент стратегии позволяет игроку совершить ограниченное количество ошибок.
-Блокировка пар: правильно сочетанные карточки блокируются и больше не участвуют в игре. Это создает динамичный игровой процесс и помогает игроку сосредоточиться на оставшихся карточках.
-Индикатор прогресса: на экране виден индикатор прогресса, который отражает текущее состояние игры. Прогрессивно закрашиваемая часть индикатора дает визуальное представление о том, сколько пар карточек осталось.

### ИНСТРУКЦИИ ПО УСТАНОВКЕ:

1. Склонируйте репозиторий на вашем компьютере.
2. Установите все зависимости, выполнив команду yarn install.
3. Запустите приложение с помощью команды yarn start.
